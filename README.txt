En este proyecto haermos una validacion de email, donde estaremos utilizando las expresiones regulares o mejor conocida como regex
(regular expression), de un email, si el email ingredado por el usuario no tiene el cuerpe que nos pide la expresión reglar 
este email será inválido, y le mostrará un mensaje en pantalla al usuario.

De igual forma si no llena ninguno de los campos pedidos, se le mostrará la alerta para que los llene. Y hasta que no pase la 
validación del email y llene los campos requeridos no podrá enviar su correo.

Esto lo estaré desarrollando con HTML, CSS y JavaScript y publicare el repositorio con git a mi github personal.
